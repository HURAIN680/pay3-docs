Smart accounts are Soroban contracts deployed on Stellar that act as programmable spending vaults. Once you link a deployed contract to your Pay3 account, AI sessions can reference it to authorize and execute on-chain payments within your defined policies. This endpoint returns every smart account you have linked, ordered by most recently linked first.

## Endpoint

**`GET /api/smart-accounts`**

Authentication is **required**. Include either the `pay3_session` cookie or an `Authorization: Bearer <session-token>` header.

---

## Response

All Pay3 API responses are wrapped in a standard envelope:

```json
{
  "data": [ ... ],
  "requestId": "req_01hx2q3r4s5t6u7v8w9x0y1z"
}
```

A successful `200 OK` response returns an array of smart account objects inside `data`. The array is empty (`[]`) if you have not yet linked any contracts.

<ResponseField name="data" type="array">
  Array of linked smart account records.

  <Expandable title="item fields" defaultOpen>
    <ResponseField name="data[].id" type="string">
      Pay3 internal smart account record ID (UUID).
    </ResponseField>

    <ResponseField name="data[].userId" type="string">
      The Pay3 user ID that owns this smart account record.
    </ResponseField>

    <ResponseField name="data[].walletId" type="string">
      The Pay3 wallet record ID this smart account is associated with.
    </ResponseField>

    <ResponseField name="data[].contractId" type="string">
      The Soroban contract address on Stellar — a 56-character `C...` string.
    </ResponseField>

    <ResponseField name="data[].network" type="string">
      The Stellar network this contract is deployed on — `TESTNET` or `PUBLIC`.
    </ResponseField>

    <ResponseField name="data[].createdAt" type="string">
      ISO 8601 timestamp of when this smart account was linked to your Pay3
      account.
    </ResponseField>

    <ResponseField name="data[].updatedAt" type="string">
      ISO 8601 timestamp of when this smart account record was last updated.
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="requestId" type="string">
  A unique identifier for this API request, useful for support and debugging.
</ResponseField>
